AAMS

A better way to track attendance

Forget hectic attendance management systems and switch to a snappy and responsive system built on top of a Universal Attendance API, which can track attendance via multiple sources of information.

Universal API access

This system comes with a API that allows for a variety of means to mark attendance: you could mark attendance with ID Card Scanners, Fingerprint sensors or via your own custom tooling.

Cost Effective

The website can be hosted serverless on AWS Lambda or Cloudflare Pages, which can keep hosting costs as low as 120 Rs a month.

Constant Uptime

The website has been developed in a way that ensures that the system has minimal server issues. This doesn't get affected by the technical issues on the UMS.

Highly customisable

This system can be altered to support different means of measuring attendance, which allows for custom-built modules that allow innovative ways of marking attendance like: Attendance marked via a quiz.
